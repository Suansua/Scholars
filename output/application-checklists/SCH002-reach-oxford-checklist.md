# Application Checklist: Reach Oxford Scholarship

> **Scholarship**: Reach Oxford Scholarship  
> **University**: University of Oxford  
> **Country**: United Kingdom  
> **Estimated Deadline**: ~February 2026 (for October 2026 entry)  
> **UCAS Deadline**: October 15, 2025 (for 2026 entry)  
> **Main Portal**: https://www.ox.ac.uk/admissions/undergraduate/fees-and-funding/oxford-support/reach-oxford  
> **Status**: ⚠️ REQUIRES VERIFICATION

---

## Critical Information

### Two-Stage Process
1. **UCAS Application**: October 15, 2025 (for 2026 entry)
2. **Reach Oxford Scholarship Application**: ~February 2026 (after UCAS)

You MUST apply through UCAS first to be considered for the scholarship.

## Part 1: UCAS Application

### UCAS Account Setup
- [ ] Create UCAS account at www.ucas.com
- [ ] Note UCAS ID number
- [ ] Link to school/college if applicable

### Course Selection
- [ ] Research Engineering Science at Oxford
  - Or other STEM course of interest
- [ ] Note UCAS course code
- [ ] Check specific entry requirements
- [ ] Check admission test requirements

### Academic Qualifications Section
- [ ] Enter ZIMSEC O-Level results
- [ ] Enter Cambridge A-Level subjects
  - If predicted: enter predicted grades
  - If achieved: enter actual grades
- [ ] Enter any other qualifications

### Personal Statement
- [ ] Draft personal statement (4,000 characters max)
- [ ] Focus on:
  - Academic interest in chosen subject
  - Relevant reading/projects
  - Skills and qualities
  - Why Oxford specifically
- [ ] Get feedback from teachers
- [ ] Finalize and proofread

### Reference
- [ ] Arrange academic reference from school
- [ ] Confirm referee has UCAS login
- [ ] Follow up to ensure submission

### Submit UCAS
- [ ] Pay UCAS fee (verify current amount)
- [ ] Submit by **October 15, 2025**
- [ ] Save confirmation

## Part 2: Oxford Admission Tests (Verify Requirements)

### Check if Admission Test Required
Different subjects require different tests. For Engineering Science:
- [ ] Check if PAT (Physics Aptitude Test) required
- [ ] Register for test
- [ ] Prepare for test (practice papers available)
- [ ] Take test (typically late October/early November)

## Part 4: Reach Oxford Scholarship Application

### Eligibility Verification
- [ ] Confirm Zimbabwe is in eligible countries list

### Scholarship Application (After Receiving Oxford Offer)
- [ ] Access Reach Oxford application form
- [ ] Complete financial information
- [ ] Write scholarship-specific statement
- [ ] Submit supporting documents

### Required Documents for Scholarship
- [ ] **Passport copy**
- [ ] **Financial documentation**
  - Family income evidence
  - Bank statements
  - Letter explaining circumstances
- [ ] **Academic transcripts**
- [ ] **Additional scholarship essay** (if required)

---

## English Language Requirements

### Oxford's English Requirements
- [ ] Check Oxford's accepted English qualifications
- [ ] **Potential Waiver Routes**:
  - Cambridge A-Level English Literature (Grade B+)
  - Cambridge A-Level English Language (Grade B+)
  - IGCSE English as First Language (Grade B+)

### If IELTS Required
- Minimum typically: IELTS 7.0 (7.0 in each component) for most courses
- Higher for some courses: 7.5
- [ ] Register for IELTS if no waiver applies
- [ ] Take IELTS (results valid 2 years)
